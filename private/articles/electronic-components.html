30 Electronic Components Your Lab Can't Miss (2022)

by Martin Pellizzer - 10/11/2022

GPS: GPS (Global Positioning System) is used to monitor the position of a device. It's basically a surface-mount chip that processes signals from GPS satellites using a small antenna.

Magnetometer: Magnetometers are used to sense the Earth's magnetic field. They are found in many handheld devices such as digital compasses, cameras, and smartphones.

Object Presence Sensor: Object presence sensors are used to detect if an object is within a preset range, but it does not measure distance. These sensors are mostly optical or magnetic.

Passive Infrared Sensor: Passive infrared motion sensors (PIR) detect object by responding passively to heat radiation. You can use them to sound alarms, switch lights, and build video camera monitoring systems.

Proximity Sensor: Proximity sensors can measure the distance between itself and another object by using (for example) infrareds, ultrasoud, and capacitance.

Linear Position Sensor: Linear position sensors measure the position of a sliding object within a stationary enclosure. For example, you can use them to build robotic arms, computer-controlled machine tools, and 3d printers.

Rotary Position Sensor: Rotatory position sensors measure the orientation of a rotating device. You can use it solar array positioning, remotely piloted vehicles, antenna positioning, etc...
